# 42KL Python Discovery Piscine

This repository contains the projects and exercises completed during the Python Discovery Piscine at 42 Kuala Lumpur.

The exercises focus on learning Python fundamentals through small problem-solving tasks and practical scripting exercises.

## Modules

- Module0 – Introduction and first scripts
- Module1 – Variables and user input
- Module2 – Conditions
- Module3 – Loops
- Module4 – String manipulation
- Module5 – Lists and collections
- Module6 – Arrays and command-line parameters

## Topics

- Variables
- Input / Output
- Conditions
- Loops
- Strings
- Lists
- Sets
- Command-line arguments
- Problem solving
